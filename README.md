# MIDILOOPER

This code is in development.

## Build instructions

    mkdir <BUILD_DIR>
    cd <BUILD_DIR>
    cmake <MIDILOOPER_DIR>
    make

## Features

* Loop Midi tracks over "Alsa sequencer" or "Jack"
* Midi track editor
* Midifile reader and writer
* Remote keyboard and midi to mute unmute tracks
* Record midi events
